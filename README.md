# 🌍 Agência de Viagens Vitorino

Olá! 👋  
Esse é o site da **empresa de viagens Vitorino**, desenvolvido com o objetivo de apresentar destinos incríveis de forma simples, moderna e atrativa.

---

## ✨ Sobre o projeto

O site foi criado para simular uma agência de turismo, onde os usuários podem explorar diferentes tipos de destinos ao redor do mundo.

A ideia foi construir algo visualmente bonito e organizado, aplicando conceitos de HTML e CSS na prática.

---

## 🚀 O que o site possui

- Página inicial com destaque principal
- Seção de destinos populares
- Página extra com mais destinos
- Área "Sobre nós"
- Formulário de contato
- Mapa ilustrativo de localização

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 💻 Como Visualizar o Site 

Mesmo que o link não abra direto, você pode rodar o site localmente:

### 1️⃣ Baixar ou clonar o projeto
Se você tiver o Git instalado:
```bash
git clone https://github.com/EllenVitorino/viagemVitorino.git

### Como abrir o site (passo a passo)

1. Verifique a estrutura de pastas:
   - index.html
   - destinos.html
   - style.css

2. Abrir a página:
   - Abra a pasta do projeto.
   - Clique duas vezes em `index.html` para abrir no navegador.

3. Atualizar após editar:
   - Salve o arquivo no editor.
   - Recarregue o navegador com F5.
   - Force recarregamento com Ctrl+F5 se necessário.

4. Testar em celular:
   - Abra Ferramentas do Desenvolvedor (F12) → ícone de dispositivo → escolha um modelo.

5. Usar servidor local (opcional):
   - Com Python: `python -m http.server 8000` → abra http://localhost:8000
